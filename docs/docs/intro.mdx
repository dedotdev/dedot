# Introduction

Dedot is the next-generation JavaScript client for Polkadot and Substrate-based blockchains.
Designed to elevate the dapp development experience, Dedot is built to be lightweight and tree-shakable, offering precise Types and APIs suggestions for individual Substrate-based blockchains and ink! Smart Contracts.
Dedot also helps dapps efficiently connect to multiple chains simultaneously as we head toward a seamless multi-chain future.

## Features

<ul>
  <li>Small bundle size, tree-shakable</li>
  <li>Typed APIs for on-chain interactions</li>
  <li>Typed ink! Contract APIs</li>
  <li>Build on top of both the new & legacy (*deprecated soon) JSON-RPC APIs</li>
  <li>Support light clients (e.g: [smoldot](https://github.com/smol-dot/smoldot))</li>
  <li>Native TypeScript type system for scale-codec</li>
  <li>Compatible with [@polkadot/extension](https://github.com/polkadot-js/extension)-based wallets</li>
  <li>Fully-typed low-level JSON-RPC client</li>
  <li>Familiar API-style with [`@polkadot/api`](https://github.com/polkadot-js/api)</li>
  <li>Easy & fast migration!</li>
  <li>Support Metadata V14, V15 (latest)</li>
  <li>Metadata optimization (caching, compact mode)</li>
  <li>... and a lot more</li>
</ul>
